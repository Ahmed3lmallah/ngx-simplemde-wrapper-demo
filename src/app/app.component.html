Two-way binding

The event binding `(SimpleMDE)` ensures that changes made in the editor are reflected on the model. Of course, any method can be used onChanges rather than just setting the model=$event

Additionally, adjusting the model programmatically should be reflected on the editor. That is where the property binding `[SimpleMDE]` comes handy.

Try it out... The button below sets the model to an empty string.



Customized toolbar

Custom configuration can be applied to any editor using the directive input `[options]="customOptions"`, where customOptions is an object of type `SimpleMdEOptions`

For a full list of options, please refere to the documentation
